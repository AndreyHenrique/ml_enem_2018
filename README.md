# ml_enem_2018
Projeto de Machine Learning que analisa como fatores socioeconômicos influenciam o desempenho no ENEM. Utilizando os microdados de 2018, realizamos pré-processamento, modelagem e comparação de algoritmos (Regressão Linear, Random Forest, LightGBM e CatBoost) para estimar a nota média dos candidatos e investigar padrões de desigualdade educacional.
